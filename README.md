# 🚢 Ship Management System

This is my first backend project built with Node.js + Express.

The idea was to simulate a simple ship and port management system where ships can be created, loaded with cargo, moved between ports, and all movements are tracked in a history log.

---

## ⚓ What this project does

- Create ports
- Create ships with weight limits
- Load cargo onto ships
- Move ships between ports
- Track movement history with cargo details

---

## 🧠 What I learned

- How to build a backend from scratch using Node.js and Express
- How to create and consume a REST API
- How frontend and backend communicate using fetch
- Managing in-memory data structures
- Basic business logic (weight limits, validations, etc.)
- Structuring a simple full-stack project

---

## 🚀 How to run it

```bash
npm install
node server.js
```
Then open:
http://localhost:3000
